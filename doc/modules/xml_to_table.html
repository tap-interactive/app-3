xml_parser.read_xml (filename, category)
Finds wanted tags in a xml file and saves it to allNews table. The function searches for the tags "title", "description", "link" and "pubDate" for every news in an xml file . The text between the tags are then added to a table (news) which are then added to a table containing all news called "allNews"

Parameters:

  • filename A string corresponding to the name of an xml file.
  • category The category of the file.

Returns:

    allNews - table containing tables with information about every news, such as title, description etc.
