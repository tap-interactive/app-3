print_text (mixed_text, x, y, max_width, max_height, font, character_case)
A function that prints a string on a surface. The surface size is dependent on the length of the string as well as the restrictions of allowed maximum width and height. The parameters are:

Parameters:

  • mixed_text (The string to be printed) [String]
  • x (The x-position of the surface's upper left corner) [Integer]
  • y (The y-position of the surface's upper left corner) [Integer]
  • max_width (The maximum width the surface is allowed to occupy) [Integer]
  • max_height (The maximum height the surface is allowed to occupy, if the text is too long it ends with "...") [Integer]
  • font (The name of the font as well as the size of it) [Table]
  • character_case (The type of the character wanted to be printed [uppercase or lowercase]) [String]
