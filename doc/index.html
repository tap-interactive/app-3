surface Emulator Surface module Surface data types The surface simulation on emulator based on the Zenterio API used for operating the surface like creating a surface, get the pixel etc.
system Emulator system Module The system simulation on emulator based on the Zenterio API used for system control of system, create a new timer or return the time clock of system.
timer Emulator timer Module The timer simulation on emulator based on the Zenterio API used for time control of system.
