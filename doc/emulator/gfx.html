Functions

new_surface (width, height)
Creates and returns a new 32-bit RGBA graphics surface of chosen dimensions. The surface pixels are not initialized; clear() or copyfrom() should be used for this. An error is raised if enough graphics memory cannot be allocated.

Parameters:

  • width The width of the surface (Must be a positive integer and less than 10000.)
  • height The height of the surface (Must be a positive integer and less than 10000.)

Returns:

    surface The new surface with the width and height set
update ()
gfx.update() Makes any pending changes to screen visible. The surface that shows up on the screen when gfx.update() is called. Calling gfx.set_auto_update(true) makes the screen update automatically when screen is changed (for development; too slow for animations)
loadpng (path)
gfx.loadpng(path) Loads the PNG image at into a new surface that is returned. The image is always translated to 32-bit RGBA. Transparency is preserved. A call to surface:premultiply() might be necessary for transparency to work. An error is raised if not enough graphics memory can be allocated.

Parameters:

  • path The source path used to load the png image

Returns:

    surface The surface of the new loaded png image
loadjpeg (path)
gfx.loadjpeg(path) Loads the JPEG image at into a new surface that is returned. The image is always translated to 32-bit RGBA. All pixels will be opaque since JPEG does not support transparency. An error is raised if not enough graphics memory can be allocated.

Parameters:

  • path The source path used to load the png image

Returns:

    surface The surface of the new loaded png image
