Surface:get ()
Surface:get() Copy pixels from one surface to another, using hardware acceleration. Parts or all of can be copied. Scaling is done if dest_rectangle is of different size than src_rectangle. Areas outside of source or destination surfaces are (will be) clipped. If is nil, the whole is used. If is nil or omitted, x=0, y=0 are assumed and width and height are taken from . If doesn't specify width or height, these values are also taken from . If is true, copying is blended using the alpha information in . If false, the alpha channel is replaced by the values in . Default is false.

Surface:destroy ()
Frees the graphics memory used by this surface. The same is eventually done automatically by Lua garbage collection for unreferenced surfaces but doing it by hand guarantees the memory is returned at once. The surface can not be used again after this operation.
